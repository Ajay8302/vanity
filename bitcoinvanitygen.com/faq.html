In some way it is a bit like having a personalised number plate on your car.

What is Paper Wallet?
A paper wallet is a mechanism for storing bitcoins offline as a physical document that can be secured like cash or anything else of real-world value. Paper wallets are generally created by printing a new public address and private key onto paper, and then sending bitcoins from a "live" wallet to the printed wallet's public address for safekeeping. If good security practices are followed, paper wallets are one of the safest ways to to store Bitcoins. BitcoinVanityGen.com is including paper wallet to all created vanity addresses.

How to import a Private Key into a Wallet?

Blockchain.info (easy) Bitcoin Core (medium) MultiBit (medium) Electrum (easy)

How to import a private key into Electrum

The easiest way:

  1. Follow basic configuration after install
  2. Click Wallet --> Private Keys --> Import --> Click YES
  3. Paste your private key and click Import
  4. Wait for synchronization
  5. Done

How to import a single private key into Bitcoin Core

Debug Window

  1. Open Debug Window
  2. Go to menu: /Help/Debug Window
  3. Click on the tab - Console

How to import a single private key into MultiBit

The easiest way:

  1. Create a new wallet saved to a file, say, example.wallet.
  2. Export the private keys, WITHOUT A PASSWORD, to a file example.key.
  3. Open up the key file in a text editor. There is a big comment at the top that describes the format. It is simply lines, each line containing the sipa format key - beginning with 5 - and the key creation date in a particular format. You will have an example in there to copy as a new wallet has one key in it.
  4. Add in your new key and a date a bit before when you created it.
  5. Import the example.key file back into the wallet.
  6. MultiBit will then replay the blocks to get the transactions and work out the balance.

How to import your private key into Blockchain.info normal wallet

The easiest way:

  1. log into your blockchain.info normal wallet account
  2. click Import / Export
  3. click Import
  4. paste your private key into Import Private Key box
  5. click Add Private Key
  6. click Import Directly

How to import a single private key into Blockchain.info *new* wallet

The easiest way:

  1. log into your blockchain.info *new* wallet account
  2. click SETTINGS
  3. click Addresses
  4. click Manage Addresses button next to Imported Addresses
  5. click Import Address button
  6. paste your private key
  7. click Import
